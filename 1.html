[D3 In 20 Days] Day 1: Scatter Plot

d3.extent() will return the [min, max] from the target object list.

The code that generate the picture: 

	
const data =[{value: 0,x: 50,y: 13,}, 
	{value: 3,x: 150,y: 63,}, 
	{value: 5,x: 250,y: 93,}, 
	{value: 7,x: 100,y: 313,}, 
	{value: 9,x: 150,y: 213,}]
const bound = d3.extent(data,d=>d.value)
const linear_scale = d3.scaleLinear()
			.domain(bound)
			.range([10,100])
// plot
const svg = d3.select('p').append('svg')
		.attrs({width:'100%',height:'100%'})
svg.selectAll('circle')
	.data(data).enter()
	.append('circle')
		.attrs({
			cx:d=>d.x,
			cy:d=>d.y,
			r:d=>{return linear_scale(d.value)},
			fill:'navy',
			opacity:0.5
		})
